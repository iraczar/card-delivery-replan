# Автотесты для формы заказа доставки карты

[![Java CI with Gradle](https://github.com/iraczar/card-delivery-replan/actions/workflows/gradle.yml/badge.svg)](https://github.com/iraczar/card-delivery-replan/actions/workflows/gradle.yml)

Автоматизированное тестирование функции перепланирования встречи для формы заказа доставки банковской карты.

## Технологии
- Java 11
- Gradle
- Selenide
- JUnit 5
- Faker
- Lombok
- GitHub Actions (CI/CD)

## Описание
Проект содержит автотесты для проверки функциональности перепланирования встречи в форме заказа доставки банковской карты.
